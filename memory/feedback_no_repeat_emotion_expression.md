name: 같은 문단 내 감정 표현 동일 단어 2회 이상 사용 금지
description: DM·문서에서 같은 문단(또는 인접 문장) 안에 동일한 감정 표현(嬉しい/嬉しく存じます/ありがたい 등) 2번 이상 반복 금지. 다른 표현으로 교체.
type: feedback
---

같은 문단 또는 인접 문장 안에 **동일한 감정 표현 단어를 2번 이상** 사용하지 않는다.

**금지 예시 (NG):**
```
お知らせいただき大変嬉しく拝見いたしました。
〜良い影響をもたらすことができたようで、大変嬉しく存じます。
```
→ 「嬉しい」가 인접 문장에서 2번 반복 = NG

**대체 패턴 (OK):**
- 첫 문장은 「ご丁寧にお知らせいただきありがとうございます」 「ご報告ありがとうございます」 「拝見いたしました」 등 감사·확인 표현
- 두 번째 문장에서 「大変嬉しく存じます」 「光栄に存じます」 등 감정 표현 1회 사용
- 「嬉しい / ありがたい / 感謝 / 光栄」 등 비슷한 계열도 같은 문단 안에서는 1회로 제한

**Why:** 2026-04-29 ちえ様 케이스. 같은 문단에 「嬉しく拝見」+「嬉しく存じます」 두 번 사용 → 세은 「우레시 우레시 하면 이상하잖아」 질책. 일본어 비즈니스 톤에서 동일 감정 표현 반복은 어색하고 깊이가 없어 보임.

**How to apply:**
- DM 작성 후 본인이 한번 읽어서 같은 감정 단어가 인접해 등장하는지 체크
- 嬉しい / ありがたい / 感謝 / 光栄 / 幸い 등 감정·감사 계열은 한 문단 1회 원칙
- 반복 발견 시 한쪽을 사실 확인 표현(「拝見いたしました」「ご報告ありがとうございます」「承知いたしました」)으로 치환
- 다른 형용사(素敵/素晴らしい 등)도 같은 문단에서 반복 X
