IG 주간 기획안 20개는 **전부 카루셀(carousel) 포맷**으로 작성한다. 릴스/단일이미지 금지.

**Why:** 2026-04-27 세은 지시. 정보형 콘텐츠 보존·공유 가치 극대화 + 슬라이드 구성으로 깊이 있는 정보 전달이 IG 운영 방향. 영상은 트위터/숏폼 영역과 분리.

**How to apply:**
- `tools/plan_weekly_content.py` 실행 시 모든 항목 format=carousel로 강제
- 엑셀 col 3(포맷) 값은 "카루셀"로만 채움
- **비주얼 방향(col 10/11)은 반드시 5슬라이드 구성으로 작성**: S1 표지 / S2 훅·문제 제기 / S3 핵심 정보1(비교표·한눈표) / S4 핵심 정보2(단계·체크리스트) / S5 CTA
- "포맷만 카루셀로 바꿀까요? 비주얼 방향까지 손볼까요?" **묻지 말 것**. 카루셀이면 비주얼 방향도 자동으로 슬라이드 단위 재작성이 디폴트
- 기존 기획안에 릴스·단일이미지가 섞여 있으면 발견 즉시 카루셀로 변환 + 비주얼 방향도 5슬라이드로 재작성
