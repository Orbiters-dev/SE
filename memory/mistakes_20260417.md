**1. dry run 안 하고 push 4번 연속**

**2. 적대감사(Harness) 안 돌림**

- 예방: 하나의 소스를 교체할 때 grep으로 해당 소스 참조 전수 확인

- 예방: 테스트 후 cleanup 스크립트 실행 또는 수동 제거 확인
