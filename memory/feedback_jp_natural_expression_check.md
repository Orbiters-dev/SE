일본어 카피/콘텐츠 작성 시 **일본 일상에서 실제 쓰는 표현**을 사용한다. 외래어·직역체로 카피 만들지 말 것.

**Why:** 2026-04-27 세은 지적. "UVクリーム" 같은 외래어 직역 표현 사용 적발. 일본에서는 「日焼け止め」가 압도적 일반 표현이고 UVクリーム은 일부 화장품 브랜드 마케팅 용어 수준. 한국식 사고로 외래어 그대로 옮기면 어색해짐. 일본 맘 타겟 콘텐츠는 SNS·일상 회화에서 실제 쓰는 표현이어야 신뢰감과 공감이 생긴다.

**How to apply:**
- 한국어 → 일본어 직역 후 반드시 "이게 일본인이 쓰는 일상 표현인가?" 체크
- 의심 가는 카타카나 표기는 한자/히라가나 표현이 메인인지 확인 (예: UVクリーム ❌ → 日焼け止め ✅)
- 모르거나 자신 없으면 Firecrawl로 일본 사이트(クックパッド·LDK·뉴스 등) 검색해서 실제 쓰임새 확인
- 잘 쓰이는 일상 표현 예시: 日焼け止め(선크림) / 虫除け(방충제) / 汗拭きシート(땀티슈) / 重ね着(겹쳐입기) / おくるみ(속싸개) / 寝かしつけ(재우기)
- 마케팅용 외래어("UVケア" "スキンケア" 등) OK, 단 카테고리·키워드 용도일 때만. 행동·아이템 묘사엔 일상어 사용
