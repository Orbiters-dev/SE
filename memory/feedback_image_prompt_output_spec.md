사진 위에 손글씨 주석/일러스트를 올리는 프롬프트(Gemini 등 이미지 생성 AI용)를 세은한테 넘길 때 반드시 다음을 포함할 것:

1. **"この写真をそのまま背景として残し、その上に〜を重ねた1枚の画像として出力してください"** — 원본 사진 위에 덧그려서 한 장 이미지로 출력.
2. **"テキストだけで答えず、必ず画像として生成してください"** — 텍스트 답변 금지.
3. **"【出力形式】元の写真を加工した1枚の画像（文章での回答は不要）"** — 출력 형식 명시 섹션.

**Why:** 2026-04-23 PPSU 스트로머그 주석 프롬프트를 만들면서 이 부분이 빠져서, Gemini가 이미지 대신 텍스트 설명만 반환. 세은이 "그림 위에 써 달라 해야 하는데 그냥 글시로만 주잖아 ㅡㅡ 장난하나"로 질책.

**How to apply:**
- "사진 위에 글씨", "사진에 주석", "이미지 덧그리기" 류 요청을 받으면 위 3개 지시문을 반드시 포함.
- 원본 묘사/텍스트 규칙 블록만 있으면 AI는 텍스트 답변으로 착각. 출력 타입 단언이 필수.
