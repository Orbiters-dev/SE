name: 계약서 이름은 한자 원칙
description: 일본인 인플루언서 계약서 フルネーム은 한자 표기로. カタカナ/히라가나만 받으면 한자 확인 재요청
type: feedback
---

일본인 인플루언서 계약서에 기재되는 フルネーム은 한자 표기를 원칙으로 한다. カタカナ/히라가나만 받은 경우 한자 표기를 재확인한다.

**Why:** 2026-04-20 りぃ様이 "ナルセリサ" 카타카나로만 제출해서 계약서 생성했다가 세은이 "이름 한자로 알려달라 그래" 지적. 일본 계약서 실무상 한자 표기가 표준이며, 한자 이름이 없는 경우에만 예외적으로 카타카나 허용.

**How to apply:**
- 계약 정보 수령 시 フルネーム이 カタカナ/히라가나뿐이면, 계약서 생성 전에 먼저 한자 확인 DM 발송
- 문구 예시: 「恐れ入りますが、契約書に記載させていただくため、ご氏名の漢字表記をお知らせいただけますでしょうか。（漢字表記がない場合は、このままカタカナで問題ございません）」
- "한자 없으시면 카타카나 그대로도 OK" 단서 포함 (외국인·선호 케이스 대비)
- 바이오가 일본어 네이티브면 한자 표기 있을 확률 99%라 적극 확인
