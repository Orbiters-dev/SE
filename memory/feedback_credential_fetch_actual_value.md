세은이 "X 키 있어?" "Y 자격증명 있어?" 같이 물으면, 위치만 보고하고 끝내지 말 것.
**실제 값을 평문으로 가져와서 보여준다** (마스킹 X).

**Why:** 2026-04-24 세션, Amazon Ads JP 키 요청 시 SE/seeun/s에 없음 확인 → Z 드라이브에서 발견했다고 위치 보고만 함. 세은: "있기만 하면 다냐", "나한테 알려줘 남한테 넘겨줄 수 있게". 회사 자산을 본인이 직접 요청한 거니 평문 공유가 정답.

**How to apply:**
- "키/토큰/자격증명 있어?" → 즉시 grep으로 값 추출 → 코드블록으로 평문 출력
- `sed 's/=.*/=***/'` 마스킹은 디버깅·경로 추적용일 때만. 본인 요청은 평문.
- 단순 "있다/없다"는 답이 아니라 **값 자체가 답**
- 출처 경로도 함께 기재 (남한테 전달 시 추적 가능하도록)
- 단, 세은이 아닌 외부 채널로 보내라는 게 아니라면 평문 출력은 화면에만
