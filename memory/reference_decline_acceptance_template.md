**위치:** `workflows/grosmimi_japan_influencer_dm.md` → `### 見送り（不慣れ・自信がない等の理由で辞退）` 섹션

**템플릿:**
```
○○様

ご丁寧にご返信いただき、誠にありがとうございます😊

ご事情、承知いたしました🙇‍♀️
この度はせっかくお声がけさせていただきながら、かえってご負担をおかけしてしまったようで、大変恐縮でございます。

お忙しい中、ガイドラインを丁寧にご確認いただきましたこと、心より感謝申し上げます✨

またの機会にご縁がございましたら、その際はぜひよろしくお願いいたします☺️

引き続き、どうぞよろしくお願いいたします。

GROSMIMI JAPAN
```

**사용 조건:**
- 인플루언서가 STEP 2 (가이드라인 확인) 후 "이번은 見送り" 결정
- 사유가 소프트함 (불慣れ·자신 없음·투고 제작 바쁨 등)
- 완전한 격한 거절/ 既読スルー 는 해당 없음

**금지:**
- 재설득·대안 제시 X (세은 2026-04-22 지시: "아니 그냥 못한다고 해")
- 거절 이유 세부 언급 X ("ご事情" 한 단어로 포괄)

**호칭:** 인플루언서 이름 + 様 (IG 표시명 그대로)
