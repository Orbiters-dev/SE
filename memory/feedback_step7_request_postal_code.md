STEP 7 배송지 정보 요청 DM에서 주소 항목은 반드시 **「・ご住所(郵便番号も含めて)」** 형식으로 작성. 우편번호 포함 명시.

**Why:** 2026-04-24 成瀨様(りぃ) STEP 7 답신에 우편번호 누락. 우리 측 자동 조회 시도(zipcloud/heartrails) 모두 실패해서 회사 측 별도 확인 필요한 상태. 세은 "앞으로 주소 요청할 때 옆에 괄호로 우편번호 포함이라고 적어".

**How to apply:**
- 워크플로우 STEP 7 템플릿: `・ご住所(郵便番号も含めて)` 로 통일
- 향후 모든 STEP 7 DM에서 동일 표기
- 워크플로우 파일도 업데이트 완료 (workflows/grosmimi_japan_influencer_dm.md)
