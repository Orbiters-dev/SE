---
name: 해시태그 수정 요청은 "포함" 표현 사용
description: 해시태그 수정 DM에서 "변경" 대신 "포함해 주시면 좋겠다"로 완곡하게 요청
type: feedback
---

해시태그 관련 수정 요청 DM은 **"변경(変更)" 표현 금지**. 대신 **"포함(含めて)"** 형태로 완곡하게 요청.

**Why:** 2026-04-21 村上様 DM에서 "ハッシュタグを下記の内容にご変更いただけますと幸いです"로 작성. 세은 지적: "앞으로 해시태그에 아래 내용 포함해 주시면 좋겠다는 표현으로 바꿔". "변경" = 지금 쓴 걸 바꾸라 → 상대에게 무언가 잘못했다는 뉘앙스. "포함" = 이 내용 들어가면 좋겠다 → 부드러운 요청.

**How to apply:**
- ❌ 「ハッシュタグを下記にご変更ください/ご変更いただけますと幸いです」
- ❌ 「今後、ハッシュタグには〜」 ← "앞으로/今後"는 Claude 지시용이지 DM 본문에 넣지 말 것
- ✅ 「ハッシュタグには下記の内容を含めていただけますと幸いです」
- ✅ 「ハッシュタグに下記の内容をご記載いただけますと幸いです」
- 한국어 번역도 "변경" X → "포함해 주시면 감사하겠습니다" 형태
- 워크플로우 STEP 11 수정 요청 템플릿 쓸 때도 해시태그만은 "포함" 어투로 대체.
