STEP 12(투고 완료 후)에서 광고코드만 단독으로 요청하지 않는다. **광고코드 + 字幕なし元データ(hello@grosmimijapan.com)** 두 가지를 항상 한 통의 DM에 함께 묶는다.

**Why:** 세은 명시 지시 (2026-04-23). 二次利用 거절자라도 원본 영상은 별도로 받아둔다 (Spark Ads 운용 백업/검증·자료보관 용도로 추정). 광고코드만 요청하면 원본 영상 수집 누락이 매번 발생.

**How to apply:**
- STEP 12 DM①은 `workflows/grosmimi_japan_influencer_dm.md` 표준 템플릿 그대로 사용 (광고코드+원본영상 묶음 요청)
- 二次利用 거절·Spark Ads 거절 등 협의 상태와 무관하게 원본 영상 요청 라인 절대 빼지 않음
- 인플루언서 메모리에 "二次利用 거절"이 있어도 그건 **재가공 사용 거절**이지 **원본 데이터 수집 거절**과 다름. 분리해서 판단할 것
- 답장 톤이 인사 마무리(잘 부탁드려요 등)여도 STEP 12 미진행 상태면 광고코드+원본영상 요청은 같이 보낸다 (별도로 나중에 보내지 않는다)
