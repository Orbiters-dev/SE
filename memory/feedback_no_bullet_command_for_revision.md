영상·하서·썸네일 등 수정 요청을 인플루언서에게 보낼 때, **「・현재 3〜5초 부분은 잘라주세요」 「・30초로 조정해주세요」** 같은 불릿+단정조로 나열하지 않는다. 대신 **한 문단 안에 흐름 있게** 적고, 마지막을 **「この方向でいかがでしょうか？」 「ご対応可能でしょうか？」** 같은 제안+질문 톤으로 마무리한다.

**Why:** 2026-04-29 今西様 케이스. 「具体的な修正点としましては、・現在の3〜5秒部分はカット ・32秒目以降カット ・30秒程度に調整」 식으로 불릿+단정조로 작성 → 세은 「싹아지 없게 말하지 마」 질책. 인플루언서가 영상 만든 사람인데 명령조로 잘라라/조정해라 하면 무례. 우리는 같이 의논하는 입장이지 지시하는 입장 아님.

**OK 예시 (세은 직접 작성한 톤):**
> 빨대 데뷔를 테마로 하기로 하고, 영상도 30초 정도로 줄이기 위해 현재 제품만 비치는 3~5초 부분이나 32초 이후의 부분들은 제외해 주셔도 되고, 전체 30초 정도로 맞추어서 저작권 프리 음악 넣으면 영상이 좀 더 템포 있게 잘 전달될 것 같은데 이런 방향으로 괜찮으실까요

**How to apply:**
- 「具体的な修正点としましては、・~ ・~」 같은 항목 나열 X
- 「~~するために、~~部分などはカットしていただいても問題ございませんので、~~いただけますと、~~かと存じますが、この方向でいかがでしょうか？」 식 한 문단 흐름
- 「カットしてください」 명령조 X → 「カットしていただいても問題ございません」 허용형
- 마무리는 반드시 「いかがでしょうか」 「ご対応可能でしょうか」 형 질문
- STEP 11 「修正あり」 워크플로우 템플릿 그대로 쓰지 말고 흐름형으로 재작성
