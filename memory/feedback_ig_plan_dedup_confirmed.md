IG 주간 기획안 생성 시 반드시 다음 방식 유지:

1. `tools/plan_weekly_content.py`의 SYSTEM_PROMPT에서 하드코딩된 "企画テーマ例" 절대 복구 금지
2. `_load_recent_titles(n_weeks=4)` — 최근 4주 Excel에서 제목+주제 로드 → 프롬프트 주입
3. 배치 간 dedup — 각 배치 생성 전 지금까지 생성된 `all_plans` 전부 프롬프트에 포함

**Why:** 2026-04-24 기존 방식이 4주 연속 같은 8개 주제(月齢別ストローマグ/PPSU比較/水分補給/韓国ママ時短 등) 돌려막기 해서 세은 분노. 수정 후 "완전 좋네 앞으로 이렇게 해" 확정.

**How to apply:**
- 매주 금요일 자동 실행 시 이 구조 유지
- 하드코딩 예시 다시 넣지 말 것 (Claude가 예시만 앵무새처럼 반복함)
- 과거 주와 다른 새 각도 · 새 시나 · 새 월령으로 생성해야 함
