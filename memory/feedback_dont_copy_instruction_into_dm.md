name: 세은 지시 문구를 DM 본문에 그대로 복붙 금지
description: 세은이 "앞으로 이렇게 해" 등 Claude용 지시를 말하면, 지시 대상어(앞으로/이번부터 등)는 DM에 쓰지 말 것
type: feedback
---

세은이 Claude에게 행동 지시를 줄 때 쓰는 메타 표현("앞으로", "이번부터", "~식으로 해", "이렇게" 등)을 DM 본문 문구로 복사해서 넣지 말 것. 지시 **방향**을 파악해서 상황에 맞는 **DM 문장**으로 변환해야 한다.

**Why:** 2026-04-21 村上様 DM에서 세은이 "앞으로 해시태그에 아래 내용 포함해 주시면 좋겠다는 표현으로 바꿔"라고 말함. 여기서 "앞으로"는 "Claude 너 앞으로는 이렇게 표현해" 지시였는데, 내가 DM 본문에 「今後、ハッシュタグには〜」로 그대로 복붙. 세은 지적: "앞으로는 왜 붙여. 그건 니한테 앞으로 그렇게 하라고 말한 거고".

**How to apply:**
- 세은의 지시문에서 Claude를 향한 메타 표현은 제거하고 의도만 반영.
  - "앞으로 이런 식으로 해" → DM에 "앞으로/今後" 붙이지 마, 그냥 해당 표현만 쓰기
  - "이번부터 ~로 바꿔" → "이번부터" 빼고 표현만
  - "~식으로 ~하라고 말해" → "말해" 같은 메타 동사 떼고 의도만
- 세은 말을 받으면 두 번 읽기: (1) 나한테 주는 지시인가? (2) 인플루언서에게 보낼 문구인가?
- Claude용 지시 키워드: "앞으로", "이번부터", "~식으로 해", "~하라고 말해", "이렇게 해"
- 인플루언서용 문구 키워드: 따옴표로 직접 인용("~", 이런 식으로 말해) 또는 완결된 문장 그대로.
