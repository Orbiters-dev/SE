인플루언서가 **아직 제작·결과물이 없는 시점**(수락만 한 단계)에 「嬉しく拝見いたしました」 「素敵なご投稿」 같은 결과물 칭찬 표현은 사용 금지. 표현이 어색하다.

**금지 시점 + 표현:**
- 수락 응답 → 「嬉しく拝見いたしました」 X (아직 본 게 없음)
- 수락 응답 → 「素敵なご制作になりそうですね」 X (아직 만든 게 없음)
- 단순 동의·합의 → 「とても嬉しく存じます」 단독 X (감정 과잉)

**대신 사용:**
- 제안·테마를 받아들여 줬을 때 → **「ご快諾いただき、誠に感謝申し上げます」**
- 협조·조정에 응해줬을 때 → **「ご協力に感謝申し上げます」** / **「お受け入れいただきありがとうございます」**
- 진행 동의 → **「進めていただけるとのこと、何卒よろしくお願いいたします」**

**Why:** 2026-04-29 遠藤様 케이스. 「メインテーマでのご制作を進めていただけるとのこと、嬉しく拝見いたしました」로 작성 → 세은 「아직 제작도 안 했는데 뭘 기쁘게 봐 표현 이상. 협조해 주셔서 감사합니다 / 받아들여 주셔서 감사합니다 이런 식으로」 질책.

**How to apply:**
- DM 작성 전 인플루언서 단계 점검: 「본 것이 있나? 만든 것이 있나?」
- 결과물(영상/사진/투고) 수신 후 → 「素敵な〜」 「拝見いたしました」 OK
- 수락·합의·일정 조정 응답 후 → 「ご快諾」 「ご協力」 「お受け入れ」 톤
- 「嬉しく」「光栄」 등 감정 표현은 결과물 또는 명확한 긍정 사실에만 사용
