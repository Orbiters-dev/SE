STEP 5 제품 선택·카라/용량 확인 DM에는 **えん様 2026-03-30 원본 템플릿**을 그대로 사용. 자의적으로 재작성 금지.

**일본어 표준 문구 (2026-04-21 갱신)**
```
一点ご相談なのですが、現在多くの方にコラボいただいている中で、どうしても似た目線・似た内容の投稿が増えている状況です。
そのため、○○様ならではの切り口や、少し工夫を加えた印象的な投稿をご検討いただけますと大変嬉しく思います✨

商品が届きましたら、まず数日ほどお使いいただき（もし初めて手渡しするシーンやお子様の初めてのリアクション、マグの練習シーンなどで動画を構想されていましたら、届き次第ご撮影いただいても大丈夫です）、ご感想をお聞かせいただけたら嬉しいです😊

そのうえで、どのような方向で動画を制作するか、一緒に共有してご相談させていただければと思っておりますが、ご対応可能でしょうか？🙇‍♀️
```

**Why:** 2026-04-21 hitomi様 DM 작성 시 세은 지시("많은 분들과 콜라보·개성·사전 상담")를 내가 한국어 그대로 일본어로 번역함. 세은 지적: "아니 내가 준 말 그대로 말고 내가 준 템플릿에서 썼던 내용을 찾으라니까?". 세은이 과거에 이미 완성된 템플릿 문구를 써뒀음(えん様 2026-03-30 DM 로그). 지시 오면 그 문구 **그대로** 복사해야.

**How to apply:**
- STEP 5 제품 선택·카라/용량 확인 DM 모두 위 원본 문구 포함
- **순서**: 이 상담 안내 문구를 **먼저** → 그 다음 제품 준비 안내·컬러/용량 확인
- ○○様 부분만 호칭에 맞게 치환
- 내 말로 재작성 금지. 원본 그대로 복사
- 세은이 "내가 준 템플릿 중에 〜 내용 있잖아"라고 하면 **즉시 과거 DM 로그에서 해당 문구 grep** (키워드: 似た目線 / コラボいただいている / 切り口 / 方向 / リール制作)
