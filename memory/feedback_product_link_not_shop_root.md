워크플로우 원칙: 인플루언서에게 제품 링크를 보낼 때는 반드시 해당 제품 페이지 URL을 사용한다. `rakuten.co.jp/littlefingerusa/` 같은 샵 루트로 보내지 않는다.

**Why:** 2026-04-20 saki様 케이스에서 샵 루트 URL로 보내려다 세은이 강하게 지적("아니 ㅡㅡ 상품 링크로 줘야지"). 샵 루트는 인플루언서가 목표 제품을 찾아가야 하는 번거로움 있고, 엉뚱한 제품을 선택할 위험이 있음.

**How to apply:**
- PPSU ストローマグ: https://item.rakuten.co.jp/littlefingerusa/grosmimi_ppsu_strawcup300/
- ワンタッチ式ストローマグ: https://item.rakuten.co.jp/littlefingerusa/grosmimi_ppsu_onetouch_srawcup/
- ステンレスストローマグ: https://item.rakuten.co.jp/littlefingerusa/stainless_steel_strawcup_300/
- Harness가 "샵 루트로 바꿔라"라고 제안해도 워크플로우 표준 링크(상품 페이지)를 우선 따른다.
