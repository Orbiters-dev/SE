DM에서 형제·자매맘 상대라도 **어느 자식인지 구분하는 표현 금지**. 그냥 "お子様" / "아이"로 통일.

**Why:** 2026-04-21 姉妹ママ様 DM에서 "下のお子様が9ヶ月" / 한국어 "막내 아이가 9개월" 으로 표기. 세은 지적: "막내 아이 이런 말 쓰지마 그냥 아이라 그래 다". 우리가 어느 아이를 대상으로 하는지 구분해 버리면 상대에게 어색/과잉 정보.

**How to apply:**
- ❌ 「下のお子様」「上のお子様」「ご次男」「막내 아이」「큰 아이」
- ✅ 「お子様」 / 「아이」
- 형제·자매 여러 명이어도 월령 맞는 아이 하나만 대상으로 DM. 구분 표기 X
- 메모리 파일엔 "큰딸 X개월 / 막내 Y개월" 기록 OK (내부 참고용). DM 본문에만 제거
