---
name: 공식 색상·제품명 그대로만 사용
description: 인플루언서가 다른 이름(타 브랜드 색상·영어명·약칭)으로 말해도, 우리는 항상 라쿠텐에 저장된 공식 색상·제품명으로만 DM·계약서 작성
type: feedback
---

인플루언서가 어떤 이름·별칭으로 색상/제품을 부르더라도, **우리 측 회신·계약서는 항상 라쿠텐 공식 페이지 표기 그대로** 쓴다.

**Why:** 2026-04-22 세은 지시 — "인플루언서가 다른 이름으로 말해도 우리는 항상 저장된 이름으로만 말해야 돼". 브랜드 정체성·내부 일관성 유지. 인플루언서가 경쟁사 색상명(예: `ベアバター`는 grosmimi 스테인리스 색인데 다른 브랜드와 혼용 가능성)·약칭·영어명 쓰면 그 이름으로 회신하지 말 것.

**How to apply:**
- 색상·사이즈·제품명 확정 DM은 **항상 [product_color_size_options.md](product_color_size_options.md)로 검증 후 기재**
- 인플루언서가 「ベアバター の 300ml」 요청 → 공식 표기 그대로 「ステンレスストローマグ 300ml（ベアバター）」 로 회신
- 만약 그들이 부른 이름이 우리 제품에 없으면 → "해당 색상은 현재 ○○色 [공식명]이 있습니다" 로 안내
- 계약서 DOCX 생성 시 PRODUCTS_LIST 필드도 공식 표기 사용

**예시:**
- ❌ 인플루언서 "큐트한 올리브 초록 색으로" → "올리브 초록" 그대로 사용
- ✅ 인플루언서 "큐트한 올리브 초록 색으로" → 공식 표기 「オリーブピスタチオ」 로 변환해서 회신
