IG 카루셀 슬라이드 제목은 **6~10자 이내 핵심 어구**만 남긴다. 동사·빈도·이유는 제목에 넣지 말고 **설명 줄(2번째 줄)** 로 분리.

**Why:** 2026-04-27 세은 지시. 슬라이드 썸네일은 한눈에 들어와야 함. "日焼け止めはこまめに塗り直し" 같은 동사 포함 제목은 길어서 임팩트↓. 「日焼け止めはこまめに」처럼 키워드+부사 조합이 시인성·전달력 모두 우수.

**How to apply:**
- 제목 = 주어 + 핵심 키워드 (예: 「日焼け止めはこまめに」 / 「水分はストローマグで」 / 「上着は1枚プラス」)
- 설명 = 빈도·방법·근거 (예: 「朝＋2時間ごとに塗り直すのがおすすめ」)
- 한국어 제목도 동일하게 짧게 (예: "선크림은 자주" / "수분은 스트로컵으로" / "겉옷은 한 겹 더")
- 마무리 표현("～がおすすめ・～が安心")은 제목에서 빼고 설명으로
- **단, 동사가 빠지면 의미가 모호해지는 경우 동사는 반드시 살린다** (예: 「汗はやさしく」 ❌ 의미 불명 → 「汗はやさしく拭く」 ✅ / "땀은 부드럽게" ❌ → "땀은 부드럽게 닦기" ✅)
- 즉, "○○は△△で/に/まで" 처럼 명사·부사구로 의미가 닫히면 동사 생략 OK. 동사가 행위 핵심이면 살림
- 모든 슬라이드 제목이 비슷한 글자 수 + 비슷한 문법 패턴이 되도록 통일감 유지
