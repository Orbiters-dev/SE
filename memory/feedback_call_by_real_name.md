name: 본명 알게 되면 호칭 변경
description: 인플루언서의 한자 본명 알게 되면 SNS 닉네임 대신 姓(성)+様으로 호칭 변경
type: feedback
---

인플루언서의 한자/본명을 알게 되면, 이후 DM부터는 SNS 닉네임 대신 **姓(성)+様**으로 불러야 한다.

**Why:** 2026-04-21 成瀨리사(りぃ)様 DM 작성 시, 계약서용 한자 본명(成瀨理紗)을 이미 받아서 알고 있었는데도 이전 세션 호칭 "りぃ様" 그대로 사용함. 세은이 "이제 이름 알았으니까 이름으로 불러야지" 지적. 본명을 알고도 닉네임 유지하는 건 비즈니스 예의에 어긋남.

**How to apply:**
- 인플루언서 메모리 파일의 "호칭" 필드를 체크. 한자 본명 확인되었으면 `姓様` 사용.
- 계약서 정보(フルネーム) 수집된 시점부터 호칭을 姓様으로 전환.
- SNS 닉네임(りぃ, むすこ 등)은 본명 모를 때만 사용하는 임시 호칭.
- 일본식: 성(姓)만 + 様 (예: 成瀨理紗 → 成瀨様 / 箕輪萌華 → 箕輪様)
- 인플루언서가 자기 서명으로 닉네임 쓰더라도, 우리 쪽은 姓様으로 정중하게.
