인플루언서 호칭 결정 순서:
1. **한자 본명** 알면 → `姓様` (예: 成瀨様, 村上様)
2. 본명 모르고 **IG 표시명**만 있으면 → **IG 표시명 그대로 + 様** (장식 폰트 유지)
3. IG 핸들에서 추측 X

**Why:** 2026-04-21 @__noainangrm 호칭을 핸들에서 "noain様"으로 추측해서 잘못 사용. 세은 지적: "姉妹𝑐ℎ𝑎𝑛♡𝑚𝑎𝑚𝑎ྀི 그냥 이거 하면 되잖아". IG 표시명이 있으면 그대로 쓰는 게 정답. 핸들 일부를 잘라서 추측하는 건 오류 확률 높음.

**How to apply:**
- IG 표시명이 장식 폰트/이모지 섞여 있어도 **그대로 복사 + 様** (예: 「姉妹𝑐ℎ𝑎𝑛♡𝑚𝑎𝑚𝑎様」)
- 표시명 끝에 붙은 티베트 부호(ྀི) 등은 생략해도 OK (様 뒤에 이어지지 않게)
- 닉네임처럼 보이는 일부만 빼내는 추측 금지
- 본명(한자) 확인된 시점부터는 姓様로 전환 (feedback_call_by_real_name.md)
