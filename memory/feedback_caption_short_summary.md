IG 캡션은 기획안 내용을 전부 쓰지 말고 핵심 포인트만 짧게 요약.

**Why:** 2026-04-18 기획안 4슬라이드 내용 전부 풀어서 캡션 작성 → "내용을 다 쓰질 말라고 요약해서 써야지" 피드백.

**How to apply:** 캡션 작성 시 5~8줄 이내. 후크(1줄) + 핵심 팩트(1~2줄) + 체크마크 3~4개 + CTA(1줄) + 해시태그. 슬라이드에 이미 있는 상세 설명은 캡션에 반복하지 않음.
