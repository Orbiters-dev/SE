## 사용 시점

- **10개월 이하 아이 인플루언서가 ステンレス 또는 ワンタッチ 받고 싶다고 표명한 모든 경우** (2026-04-23 세은 명시)
- 인플루언서가 「ストローが違うのか」「ステンレスとPPSUの違いは？」「ステンレスマグも気になる」 등 빨대 차이 질문할 때
- STEP 5 제품 추천 또는 그 직후 답변 단계에서 발생
- ステンレス/ワンタッチ 모두 기본 빨대가 Stage 2(かため)이므로 동일 응대 가능

## 표준 답변 (DM 본문)

```
○○様

ご返信ありがとうございます☺️
それでは、[製品名] [容量]（[カラー]）にてご用意させていただきます🍼

ステンレスストローマグにつきましてもご関心いただきありがとうございます😊
おっしゃる通り、ストローの違いがございまして、PPSU ストローマグは Stage 1 ストローが装着されており、やわらかいためストローの練習を始められたお子様に適しております。
一方、ステンレスマグは基本的に Stage 2 ストローが装着されており、噛む力やお口の力がしっかりしてきた12ヶ月以上のお子様向けの仕様となっております。
そのため、まだお口の力が弱かったり、ストローに慣れていらっしゃらないお子様には少し難しい場合がございます🙇‍♀️

なお、PPSUとステンレスのパーツは互換性がございますので、お子様が成長されたタイミングでステンレス本体に切り替えていただくことも可能です🌱

引き続きどうぞよろしくお願いいたします。

GROSMIMI JAPAN
```

## 핵심 포인트

- **Stage 1 (PPSU 기본)**: やわらか / ストロー練習 시작 아이 적합
- **Stage 2 (ステンレス 기본)**: かため / 噛む力·口の力 단단해진 **12ヶ月 이상** 아이 용
- 입 힘 약하거나 빨대 미숙 → 어려움 가능 (12개월 미만에 ステンレス 직접 추천 X)
- 호환성: PPSU ↔ ステンレス 파츠 互換 → 성장 후 본체 교환 가능
- ❌ 「飲み口の構造自体は同じですが」 같은 음료수구 구조 비교 표현 사용 X (세은 지시)
- 「12ヶ月以上」으로 통일 (10ヶ月頃 X — `feedback_stage2_12month_for_influencer.md`)

## 변형 케이스

- **케이스 A: 제품 확정 + 차이 설명** — 위 템플릿 그대로 (확정 한 줄 + 차이 설명 + 호환성)
- **케이스 B: 제품 미확정 + 차이만 설명** — 첫 줄 「それでは、…」 부분 제외, 차이 설명 + 호환성만
- **케이스 C: 12개월 미만 아이가 ステンレス 직접 요청** — 차이 설명 + ステンレス는 12개월 이상 안내 + PPSU 우선 권유
