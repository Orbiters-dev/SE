---
name: 사용자 행동 단정 금지
description: 세은이 뭘 했는지/안 했는지 추측하지 말 것. 로그 기반으로 사실만 말하기.
type: feedback
---

"세은이 git pull 했을 때 들어온 거"처럼, 세은이 직접 하지 않은 행동을 세은 탓으로 돌리지 말 것.

**Why:** "나는 pull 안 했는데"라고 반박받음. 실제 범인은 이전 Claude Code 세션이 자동 실행한 git pull이었음.
**How to apply:** 원인 분석할 때 "누가 했다"가 아니라 "로그에 이런 기록이 있다"로 사실 기반 보고. 행위자가 불확실하면 확인 먼저.
