## 규칙

- **先ほど** (사키호도): 방금 전, 몇 시간 이내. 당일 이벤트 언급할 때.
- **先日** (센지츠): 며칠~몇 주 전. 당일보다 과거일 때.
- **이전に** (이젠니): 막연히 "전에"

**Why:** 2026-04-21 umi_kotetsu様 리치아웃에서 영상 올린지 1시간밖에 안 됐는데 "先日投稿されていた" 썼다가 세은 교정. 너무 시간 차가 느껴지는 표현.

**How to apply:**
- 상대방이 방금~당일 올린 콘텐츠 언급 → `先ほど投稿されていた`, `本日のお写真を拝見し`, `今朝アップされていた`
- 며칠 전 올린 콘텐츠 → `先日投稿されていた`
- 시점 불확실하면 세은에게 확인하거나 구체적 시점 묻기
