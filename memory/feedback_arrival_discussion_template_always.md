세은이 "상품 도착 후 사용 → 저희랑 영상 방향 상의" 흐름을 꺼내면 **무조건 아래 STEP 5 원문 括弧 포함 문구를 그대로 사용**.

**원문:**
```
商品が届きましたら、まず数日ほどお使いいただき（もし初めて手渡しするシーンやお子様の初めてのリアクション、マグの練習シーンなどで動画を構想されていましたら、届き次第ご撮影いただいても大丈夫です）、ご感想をお聞かせいただけたら嬉しいです😊
```

이후 보통 「そのうえで、どのような方向で動画を制作するか、一緒にご相談させていただければと思っております」로 이어감.

**Why:** 2026-04-22 세은 지시 — "내가 이 얘기 꺼내면 무조건 그 템플릿 꺼내서 썼으면 좋겠는데". 括弧 부분이 인플루언서 자율성 보장(첫 반응·연습 장면 즉시 촬영 OK)을 명시적으로 알려줌. 누락하면 인플루언서가 "꼭 며칠 사용 후에만 찍어야 하나?" 혼란 생길 수 있음.

**How to apply:**
- 세은이 한국어로 "상품 사용 후 상의하는 형태로 진행" 식으로 지시하면 → 일본어 원문 括弧 문구 그대로 복사
- 括弧 줄여서 쓰지 말 것 ("使用後に方向を相談" 같은 축약 금지)
- Auditor가 括弧 생략하라고 제안해도 무시
