### 2. 메타 광고 화이트리스팅 가이드 (긴 Q&A)

**상황:** 세은이 partnership ad code 사용한 화이트리스팅 광고 셋업 시도. 지선이 알려준 "Use existing post로 누르면 이전 광고 불러올 수 있다" 방식이 disabled.

**원인 파악:**
- Partnership ad code 입력 시 안내문: *"The format of this ad has been set by the partnership ad code"*
- 즉 ad creative source가 자동으로 `Use existing post`로 잠김 → 다른 옵션(Create ad, Creative Hub mockup) 선택 불가
- 지선 방식은 partnership ad code 안 쓰는 일반 광고 또는 partner content 그리드에 인플루언서가 권한 부여한 게시물이 떠있을 때만 가능

**세은이 결국 한 방식 (정상 셋업):**
- partnership ad code 입력 **없이** 새 광고 생성
- Ad creative → Use existing post → **partner content 그리드에 인플루언서 영상이 떠있어서 직접 클릭**
- 이건 정확히 use existing post 방식의 정석. 인플루언서 측 partnership ad permission이 ON이라 영상이 partner content 목록에 노출됨

**Thumbnail 이슈:**
- 새 광고만 인플루언서 페이지 프로필 사진이 thumbnail로 떴음
- 원인: 광고가 **Processing 상태** (active 아님) → ad creative thumbnail 아직 generation 안 됨 → 페이지 프로필 fallback
- Active 전환 후 정상 영상 썸네일로 변경됨 ✓
