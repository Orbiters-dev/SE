## アウトリーチ カスタマイズ必須ルール

**最初のDM・メール送信時、必ずクリエイターの最近コンテンツを参照し、1〜2文の個別メッセージを含めること。**

### 禁止（NG）
プロフィール基盤の汎用メッセージ。具体的なコンテンツ言及なしの定型文。

> ❌ 「投稿を拝見し、雰囲気がブランドととてもよく合っていると感じ」
> → 적은티바로남. 모든 인플루언서에게 동일한 문구는 역효과.

### 必須（OK）
transcript（動画書き起こし）またはCI分析データ（key_message, emotional_tone）を基に、そのクリエイターの最近投稿の具体的内容に言及する。

> ✅ 「最近の育児グッズ整理術の投稿を拝見しました。お子さまとの日常を丁寧に共有されている姿勢に共感いたしました」

> ✅ 「赤ちゃんの離乳食レシピ動画、とても参考になりました。食事の時間を大切にされているお姿が素敵です」

### データソース
CRM Dashboard → Influencer Sheet → 各クリエイターのCI分析結果を確認：
- **transcript**: 動画の書き起こし（Whisper）
- **key_message**: CI分析から抽出されたコンテンツの核心メッセージ
- **emotional_tone**: コンテンツの感情トーン
- **persuasion_type**: コンテンツのスタイル（educational, personal_story, humor等）

### DM自動生成時
Dashboard「Generate Drafts」ボタンから生成する場合、transcript + CI分析データ（ci_context）が自動的にn8nワークフローに送信される。n8n側のAIプロンプトでこのデータを活用し、個別カスタマイズされたオープニングを生成すること。


**報酬交渉フロー：**
1. インフルエンサーの提示額を確認
2. ユーザーに報告・相談
3. フォロワー数をもとに逆算して代替金額を提案
4. 再交渉が来たら → 最初の提示額を維持して丁重に断る

**보수 역산 공식：**
> 1. 유저가 제안할 **목표 금액을 먼저 결정** (세은에게 반드시 확인)
> 2. 목표 금액 / 팔로워 수 = 단가 X 역산
> 3. **팔로워 수 x X円 <= 목표 금액**이 되는 X 선택
> 4. DM에 **「フォロワー数 x X円」** 형식으로 계산 근거 명시
> 5. 계산 결과를 반올림해서 최종 제안
>
> ⚠️ **1.5円은 고정 기준이 아님. 매번 목표 금액에서 역산할 것.**

## 特殊ケース対応

### リール制作が負担・忙しいと言われた場合
→ 무리하지 않아도 된다고 완곡하게 전달.

### タイミングが合わない（忙しい・子どもの月齢が早い等）
→ 담백하고 간결하게 클로즈. 나대는 표현 사용 금지. 이쪽에서 다시 연락한다고 전달.

### 完全に断られた場合
→ 返信不要。

### ホワイトリスティング拒否
→ 契約しない。

### 既読スルー
→ フォローアップなし。

### 報酬交渉
→ ユーザーに報告し確認してから返信。自分で金額を決めない。

## 注意事項

- **DM 초안을 유저에게 전달할 때는 반드시 일본어 원문 + 한국어 번역을 함께 제시한다**
- **유저가 직접 작성한 템플릿을 보내주면 그 말투·표현·구조를 그대로 학습하여 이후 DM에 반영한다**
- 返信文は必ず冒頭に「○○様」を入れる（名前不明の場合はユーザーに確認）
- 報酬金額は必ずユーザーに確認してから提示する
- **transcript 없이 프로필 기반 이메일 절대 금지 — "적은티바로남"**
- PPSU・漏れ防止・洗浄に関する表現は核心製品ポイントの ガイドラインを必ず守る
- 競合ブランドの名前を絶対に出さない
- **タイミングが合わない場合、「投稿楽しみにしています」などビジネスの範囲を超えた表現は使用禁止**
- **1つのDM内で同じ絵文字を複数回使用しない**
- **製品選定・推薦のDMを送る際は、必ず該当製品のリンクを本文に添付する**
