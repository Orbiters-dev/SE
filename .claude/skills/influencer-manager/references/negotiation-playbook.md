GROSMIMI JAPAN 인플루언서 협상 가이드

## 1. 팔로워 기준 협업 유형

| 팔로워 수 | 기본 제안 | 최대 조건 |
|-----------|-----------|-----------|
| ~5,000 | 기프팅만 | 기프팅만 |
| 5,000~10,000 | 기프팅 | 기프팅 + 추가 제품 |
| 10,000~30,000 | 기프팅 우선 → 유료 협의 가능 | 기프팅 + ¥10,000 |
| 30,000~100,000 | 유료 (¥15,000~30,000) | ¥50,000 + 기프팅 |
| 100,000+ | 유료 (¥30,000~) | 담당자 직접 협의 |

> **주의**: 금액은 담당자 승인 없이 절대 제시 금지

## 2. 기프팅 조건 (기본)

**제공 제품:**
- PPSU 240ml 1개 + 빨대 세트 또는
- One-touch 240ml 1개 또는
- 아이 월령에 맞는 담당자 추천 제품

**납품물 (최소 요건):**
- Reels 1개 (최소 15초, 최대 90초)
- Story 2~3개 (제품 사용 장면 포함)

**해시태그 필수:**
- #grosmimi #グロスミミ #PR #商品提供

**게시 기한:** 제품 수령 후 14일 이내

## 3. 유료 협업 조건 (10k+ 팔로워)

**기본 제안:**
```
Reels 1개 + Story 3개 = ¥15,000 + 기프팅 제품
```

**협상 레버:**
- 납품물 추가 (Story +1개) → +¥3,000
- 화이트리스팅 포함 → +¥5,000~10,000
- 독점 기간 연장 (30일→60일) → +¥5,000

**협상 하한선 (절대 양보 불가):**
- 화이트리스팅 권한은 반드시 포함
- 해시태그 및 PR 표시 의무
- 초안 사전 검토 권한

## 4. 자주 나오는 반박 & 대응

### "보상을 더 높여줄 수 없나요?"
**대응**: 납품물을 줄이거나, 독점 기간을 단축하는 방향으로 협의 제안.
금액 자체를 올리기 전에 담당자 확인 필수.

```
日本語 예시:
「ご要望ありがとうございます。少々確認が必要ですので、
2〜3営業日以内に改めてご連絡させていただいてもよろしいでしょうか？」
```

### "화이트리스팅은 어렵습니다"
**대응**: 화이트리스팅 없으면 계약 불가. 정중하게 필수 조건임을 안내.
완전 거절 시 → Gifting 전환 고려 (단, 유료로 진행 중이었다면 계약 파기).

```
日本語 예시:
「ホワイトリスティングは弊社の契約条件に含まれており、
省略が難しい状況です。もし難しい場合は、ギフティング形式での
ご協力も検討いただけますでしょうか？」
```

### "독점 조건이 너무 깁니다"
**대응**: 30일 → 14일로 단축 가능 (담당자 승인 필요).

### "계약서가 너무 복잡합니다"
**대응**: 핵심 조항 3가지 요약 제공 (납품물, 보상, 권리).
전문적인 계약서임을 강조하되, 질문은 언제든 환영.

## 6. 계약 진행 체크리스트

계약서 발송 전 반드시 확인:

- [ ] 인플루언서 실명 확인 (계약서용)
- [ ] 이메일 주소 확인 (DocuSign 발송용)
- [ ] 팔로워 수 재확인 (스크린샷 캡처)
- [ ] 납품물 상세 합의 완료
- [ ] 보상 금액/내용 담당자 승인 완료
- [ ] 화이트리스팅 동의 확인
- [ ] 게시 마감일 합의 완료
- [ ] 독점 기간 합의 완료
- [ ] 한국어 계약서 검토 완료
- [ ] 일본어 계약서 검토 완료
- [ ] DocuSign 수신자 이메일 확인
