# インフルエンサーコンテンツ契約書テンプレート（無償 / Gifting）

> 赤テキスト = 変数フィールド → {{変数名}} で置換

---

## 変数リスト

```
{{contract_date}}           → 契約日 (例: 2026年3月15日)
{{company_name}}            → 会社名 (例: LittleFingers USA)
{{influencer_handle}}       → @インスタID (例: @sakura_life)
{{influencer_name}}         → インフルエンサー氏名 (例: 田中さくら)
{{deliverables_type}}       → 動画プラットフォーム (例: Instagram Reels)
{{brand_account}}           → 公式タグ先アカウント (例: @grosmimi_japan)
{{hashtags}}                → 指定ハッシュタグ (例: #grosmimi #グロスミミ #PR)
{{company_signer_name}}     → 署名者氏名
{{company_signer_title}}    → 役職 (例: 日本事業部)
{{company_signer_date}}     → 署名日 (例: 2026年3月15日)
```

## 契約書本文

---

**インフルエンサーコンテンツ契約書**

本インフルエンサーコンテンツ契約（以下「本契約」）は、{{contract_date}}付で、
**{{company_name}}**（以下「甲」）と、@{{influencer_handle}} として知られる {{influencer_name}}（以下「乙」）との間で締結される。

---

**第1条（契約の範囲）**

乙は、甲より無償で提供される商品（以下「本商品」）を使用し、下記内容のコンテンツを制作・投稿するものとする。

1. 動画コンテンツ 1本（{{deliverables_type}}）

2. 本商品の特長およびメリットを、乙の通常のスタイル・トーンを尊重しつつ、甲が提示するクリエイティブガイドラインおよびブランドメッセージを適切に反映した形で、自然かつ誠実に紹介すること。

また乙は以下に同意する：

- 字幕・音楽・テキスト等を含まない元動画データの提供
- 広告利用（TikTok Spark Ads・Instagramホワイトリスティング）の許可
- 本広告の実施にあたり、広告配信上必要な場合には、乙が管理・運営するInstagramアカウントとFacebookアカウントの連動に協力すること。
- Instagram に投稿する場合、甲の公式アカウントを「co-creator（共同クリエイター）」として追加すること
- 甲から指定がある場合、指定された形式でボイスオーバーや字幕、画面内テキスト（オンスクリーンテキスト）を適切に挿入すること

商品は本契約締結後、甲より乙へ発送される。

---

**第2条（成果物）**

乙は以下を履行する：

1. 本商品を使用した動画を制作し、**投稿前に甲へ下書きを提出**すること。
2. 甲は下書き受領後**5営業日以内**にフィードバックを行うものとする。
3. 乙は**最大2回まで**合理的な修正に応じるものとする。
4. 商品到着後**14日以内**に投稿すること。

※ ただし、やむを得ない事情がある場合には、甲乙協議のうえ、投稿期限を変更できるものとする。

5. 投稿時、甲の公式アカウント（{{brand_account}}）をタグ付けすること。
6. 甲より指定されたハッシュタグまたは文言（{{hashtags}}）を投稿内に含めること。

---

**第3条（権利関係）**

1. 本件コンテンツの著作権は乙に帰属する。
2. 乙は甲に対し、甲の公式SNSやWEBサイト等における再投稿・紹介を目的とした非独占的ライセンスを付与するものとする。使用期間や具体的な範囲は、双方の合意のもと柔軟に調整する。
3. また、乙が提供した元動画データおよび制作物について、甲のデザイナーがリサイズ、形式変換、字幕追加、短尺化など、本来の内容や意図を変更しない範囲での二次加工を行う場合がある。これらの加工は、乙のイメージを損なわず、誤解を招かない簡易的な編集に限るものとする。
4. 広告利用および二次利用（Spark Ads／ホワイトリスティング等）を行う場合は、事前に乙に確認の上、乙のイメージを損なわない範囲で使用するものとする。
5. 甲は乙の事前承諾なく、誤解を招く編集・改変を行わないものとする。

---

**第4条（法令遵守および広告表記）**

乙は下記に従うものとする：

1. 景品表示法、薬機法、特定商取引法など、日本の広告法令
2. SNSプラットフォームのガイドライン
3. 明示すべき広告表記（例：**#PR #提供 #広告**）
4. Spark Ads / ホワイトリスティング利用に必要な許可の付与

甲もまた関連法規に基づき適切な広告運用を行う。

---

**第5条（対価）**

1. 本契約における乙への対価は **本商品の無償提供のみ** とする。
2. 甲は本商品提供以外の金銭的報酬・手数料・費用を支払わない。
3. 乙は本商品の提供をもって、本契約に基づくすべての対価を受領したものとする。

---

**第6条（秘密保持）**

乙は、甲の事前書面による承諾なく、本契約内容（報酬含む）を第三者に開示してはならない。

---

**第7条（契約期間および解除）**

1. 本契約は、乙による成果物の投稿が完了するまで有効とする。
2. いずれかの当事者が重大な契約違反をした場合、相手方は**書面通知後7日間の是正期間**を設け、その後解除できる。

---

**第8条（責任および免責）**

1. 各当事者は、重大な過失または故意による損害についてのみ責任を負う。
2. 間接損害・特別損害・付随的損害については、いずれも責任を負わない。
3. 損害賠償責任の上限は、本商品の通常販売価格相当額とする。

---

**第9条（完全合意）**

本契約は当事者間の完全な合意を構成し、口頭または書面による従前の合意に優先する。
