### 카피 톤 가이드 (6~24개월 맘 타겟)

타겟은 **6~24개월 아이를 키우는 엄마**. **정보 전달성 + 친근함의 중간 톤**. 딱딱한 비즈니스도 X, 너무 캐주얼한 친구톤도 X.

**한 줄 정의: "옆자리 동네 언니가 알려주는 육아 정보" 톤.**

| 영역 | OK (정보+친근) | NG ① 딱딱·비즈니스 | NG ② 너무 캐주얼 |
|------|----------------|--------------------|-----------------|
| 어휘 | 日焼け止め / お水 / お出かけ / お顔 | 紫外線対策 / 水分補給 / 外出 | — |
| 어미 (JP) | 〜のがおすすめ / 〜が基本 / 〜が目安 / 〜が◎ / 〜が安心 | 〜すること / 〜の必要がある / 〜を推奨 | 〜してね / 〜していこう / 〜してあげて |
| 어미 (KO) | "~해주세요" "~좋아요" "~정도가 적당해요" "~안심" | "~할 것" "권장" "필수" "기본 원칙" | "~해봐~" "~하자!" "~하기♡" |
| 표현 | こまめに / ちょうどいい / 目安 / おすすめ / 안심 / 딱 좋아요 | 黄金ルール / 必須 / 推奨 / 適切に / 적정량 | テンション系 過剰絵文字 |

**규칙:**
- 한자어 비율 낮추기 → 히라가나 + 부드러운 외래어 활용
- **오노마토페(ポンポン·シュッ·トントン 등)는 카루셀 전체에서 0~1개만**. 남발 금지. 매 슬라이드마다 의성어 박지 말 것
- 어미는 「〜ね」「〜で」「〜してあげて」 같은 친구 톤
- "黄金ルール" "必須" "推奨" 같은 마케팅·교과서 표현 금지
- 한국어도 "~해요" "딱 좋아요" 같은 친근 존댓말. "~할 것" "필수" 금지
- 한국어도 "톡톡·칙·박박" 같은 의성어 남발 금지. JP와 매칭되는 1개만

**참조:** [memory/reference_ad_copy_tone_jp.md](../../../memory/reference_ad_copy_tone_jp.md), [memory/feedback_jp_natural_expression_check.md](../../../memory/feedback_jp_natural_expression_check.md)

### 카루셀 5슬라이드 표준 구성 (필수)

비주얼 방향 작성 시 반드시 5슬라이드 구조로 풀 것:

| 슬라이드 | 역할 | 내용 |
|----------|------|------|
| **S1 표지** | 후킹 | 메인 사진 + 제목 텍스트 오버레이 (소제목+대제목) |
| **S2 훅·문제 제기** | 공감/의문 | "○○ 모르고 있지 않나요?" "이게 정답?" 식 의문 던지기 |
| **S3 핵심 정보1** | 데이터 | 월령별·종류별 비교표, 한눈에 보는 표, 사진 비교 |
| **S4 핵심 정보2** | 실행 | 단계 가이드, 체크리스트, NG vs OK 사례 |
| **S5 CTA** | 액션 | 저장/팔로우/프로필 유도. CTA 캐치프레이즈 대표 1개 |

작성 형식: `S1表紙：[비주얼+텍스트] / S2：[내용] / S3：[내용] / S4：[내용] / S5 CTA：[메시지]`

## CTA 캐치프레이즈 규칙 (必) — 2026-04-22 갱신

모든 기획안에 **"CTA 캐치프레이즈 후보 5개"** 포함이 필수. 카루셀 마지막 슬라이드(CTA 페이지) 또는 캡션 끝에 사용.

**5개 후보는 다음 3가지 톤만 사용 (혜택훅·DM 단독 금지):**

| # | 톤 | 기능 | 액션 | 예시 |
|---|------|------|------|------|
| 1 | 호기심 어그로 + 프로필 유도 | 궁금증 한 줄 → 프로필에 답 | **프로필 링크** | 「"結局どれがいいの…？" の答え、プロフィールに全部📌」 |
| 3 | 저장 유도 | 다시 볼 가치 강조 | **저장 + 팔로우** | 「迷ったら保存🔖 月齢別ガイドはフォローで」 |
| 4 | 공감 소속감 | "같은 고민 엄마 있어요" 톤 | **팔로우** OR **공유**(예외) | 「同じ悩みのママ友いたら、そっとシェアしてあげて🤍」 |

### 액션 통일 규칙
- **메인 액션 = 팔로우 OR 프로필 링크** (기본)
- **예외**: 4번(공감 소속감) 톤 + 정보형(가이드/체크리스트) 콘텐츠 → "공유(シェア)" 액션 허용
- 쿠폰 코드, DM, 사이트 외부 방문 등 **다른 단독 액션 금지** (보조로만 가능)

### 5개 후보 분포 가이드
- 1톤(호기심+프로필) **2개** — 같은 콘텐츠 다른 angle
- 3톤(저장 유도) **1~2개**
- 4톤(공감 소속감) **1~2개**

### 카테고리별 대표 1개 선정
- brand (제품 소개/비교) → **1톤** 우선
- tips/knowledge (정보 가이드) → **3톤(저장)** 우선
- Q&A·고민 공감형 → **4톤(공감)** 우선
- 친구 공유 가치 큰 정보형(월령별·체크리스트) → **4톤(공유 변형)** 채택 가능

- **각 후보는 JP 원문 + KO 번역 병기** (캡션 규칙과 동일)
- 5개 중 대표 1개를 CTA 슬라이드(마지막 페이지) 대제목으로 제안

## 캡션 작성 규칙

기획안 PPT를 보고 캡션을 쓸 때:

- **5~8줄 이내**로 핵심만 요약. 기획안 내용 전부 쓰지 않는다.
- 구조: 후크(1줄) + 핵심 팩트(1~2줄) + 체크마크 3~4개 + CTA(1줄) + 해시태그
- 슬라이드에 이미 있는 상세 설명은 캡션에 반복하지 않는다
- 해시태그: #グロミミ #grosmimi #ストローマグ #スマートマグ (통일)
- 한국어 번역 항상 함께 작성

**캡션 예시:**
```
もうスパウトは卒業しませんか？🍼→🥤

カーペットを汚さないために作られたもの😳

ストローマグに変えると
✔️ 舌が正しい位置に
✔️ 唇・顎の発達をサポート
✔️ 言語発達にも◎

AAPDも12〜14ヶ月での
切り替えを推奨しています📋

詳しくはプロフィールから👆

#グロミミ #grosmimi #ストローマグ #スマートマグ
```

## 규칙

1. 매주 기획안은 **tips:8, brand:6, k_babyfood:3, knowledge:3 = 총 20개**
2. 기획안에는 반드시 **일본어 + 한국어 번역** 동시 포함
3. **소제목 + 대제목** 필수 (썸네일 텍스트)
4. 엑셀 메일 첨부 금지 — 로컬 저장 + "저장 완료" 알림만 발송
5. 저장 후 반드시 지정 경로에 복사 + 메일 발송 (묻지 말고)
6. Claude API 호출은 5개씩 배치 (rate limit 방지)
7. 경쟁사 분석 데이터가 있으면 반드시 기획안에 반영
8. **CTA 캐치프레이즈 후보 5개 필수** — 모든 기획안 엑셀 13열에 JP+KO 병기. 카루셀은 마지막 슬라이드(CTA 페이지)에 대표 1개 사용
