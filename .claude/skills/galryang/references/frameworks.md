# Strategic Analysis Frameworks — Quick Reference

## SWOT Analysis
- **Strengths**: 내부 역량, 차별화, 핵심 자산
- **Weaknesses**: 내부 한계, 리소스 부족, 프로세스 문제
- **Opportunities**: 외부 기회, 시장 변화, 신규 채널
- **Threats**: 외부 위협, 경쟁 심화, 규제 변화

## Porter's 5 Forces
1. **경쟁자 간 경쟁**: 시장 내 경쟁 강도
2. **신규 진입 위협**: 진입 장벽 높이
3. **대체재 위협**: 다른 카테고리/채널로의 이탈
4. **구매자 교섭력**: 고객/플랫폼의 가격 압박
5. **공급자 교섭력**: 원재료/제조/물류 의존도

## BCG Matrix
- **Star**: 높은 성장 + 높은 점유율 → 투자 유지
- **Cash Cow**: 낮은 성장 + 높은 점유율 → 수확
- **Question Mark**: 높은 성장 + 낮은 점유율 → 투자 or 철수 결정
- **Dog**: 낮은 성장 + 낮은 점유율 → 정리
