# 세션 브리핑

세션 시작 시 아래 순서로 현재 상태를 파악하고 세은에게 보고한다.

## 실행 순서

1. **최근 세션 요약 읽기**
   - `memory/` 폴더에서 가장 최근 `session_*.md` 파일을 찾아 읽는다
   - 미완료 작업이 있으면 먼저 알린다

2. **git 최근 변경 확인**
   - `git log --since="60 hours ago" --oneline --all` 실행
   - 주요 변경사항 요약

3. **인플루언서 현황 확인**
   - `memory/influencer_*.md` 파일들을 읽어서 현재 진행 중인 인플루언서별 상태 정리
   - 오늘 해야 할 액션이 있으면 알린다

4. **reflect 상태 확인**
   - `.claude/skills/reflect/.state/last-reflection.json` 읽기
   - 마지막 세션에서 감지된 신호가 있었으면 보고
