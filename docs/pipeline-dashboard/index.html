🔎 Step 0: Discover — Find New Creators

Syncly Discovery에서 브랜드 관련 콘텐츠 + 대사(캐프션)을 함께 발굴.
대사는 outreach 이메일 작성 근거로 활용됨. Execution 탭 > Discovery & Sourcing에서 실행.
Syncly Discovery (Outbound)
크리에이터 정보 (핵들, 팔로워, 인게이지먼트) + 발견 콘텐츠 (캐프션/대사 포함) 임포트.
Not Started로 파이프라인 진입. 자동 누적.
Inbound (자동)
크리에이터가 Gifting 폼 제출 → Needs Review 자동 등록.
Review Queue에서 승인/거부.
Note: Apify Content Crawl = Track 단계(D+60 모니터링). Draft에서도 Autofill(프로필 보충)으로 활용.

⚙️ Step 1: Config — Set Batch Parameters

Batch Size, HT Threshold, Feature Toggles, Human-in-Loop 설정. 매일 배치 전에 확인 → Apply Config.
Batch Size
한 번에 보낼 수. 초기 5-10, 안정화 후 20-30.
HT Threshold
R30D 100K+ = High Touch (계약+결제), 미만 = Low Touch (무료).
RAG Email Dedup
이미 연락한 사람 자동 제외.
Apify Brand Filter
27개 브랜드/리셀러 계정 자동 제외.

📝 Step 3: Generate Drafts

n8n Draft Gen WF (fwwOeLiDLSnR77E1)

📨 Step 4: Send + Email QnA HIL

① Outreach 발송
• affiliates@onzenna.com에서 발송 | 브랜드별 Gifting 폼 URL 포함
• Draft Ready → Sent | Human-in-Loop: Send Approval
② Reply Handler — 이메일 QnA 자동화 (WF: K99grtW9iWq8V79f)
Creator 답장 감지 → Replied → Claude AI 분류:
Accept
협업 수락 → Gifting 폼 링크 자동 발송
FAQ_Gap
질문 (배송/상품/보상) → FAQ DB에서 자동 응답
Normal
일반 대화 → 감사 응답 자동
Config: Email Config 탭에서 분류 프롬프트, 템플릿, FAQ 수정 가능 (PG API)
⚡ LT: Reply 분류 후 자동 발송. Accept → Gifting 폼 링크 자동 | FAQ → 자동 응답. 사람 개입 없음.
👑 HT: Reply 분류 후 Needs Review. 마케터가 답장 내용 확인 후 수동 발송. 보상/계약 논의 포함 가능.
③ Gifting 폼 제출 (1회)
Creator가 폼 작성 → 샘플 선택 + 주소 입력 → n8n Webhook → Shopify 고객 + Draft Order 자동 생성
• WF: F0sv8RsCS1v56Gkw | 1폼으로 샘플 선택까지 완결

🔍 Step 5: Review & Accept Always HIL

Accept
마케터 Accept 클릭 → Accepted
→ n8n이 5분 poll로 감지
→ PG에서 Draft Order ID 조회
→ Shopify Draft Order 자동 완료
WF: Ow5bD5keNf2HFo83 (크리덴셜 연결 후 ON)
Decline
→ Declined | 팔로워 부족, 브랜드 불일치 등 | Note 기록
1-폼 파이프라인: Creator가 Gifting 폼에서 샘플까지 선택 완료 → Draft Order 이미 생성됨 → Accept 시 바로 오더 나감. Gifting2 폼 없음.
⚡ LT: Accept 클릭만으로 완료. 계약/보상 없음. 무료 샘플 제공.
👑 HT: Accept 전에 계약서 발송 + 서명 확인 필요. 보상 조건 합의 후 Accept. 결제는 콘텐츠 확인 후 처리.
판단: Content 링크(IG/TT) → 팔로워, 콘텐츠 품질, 브랜드 적합성, 아이 나이

📦 Step 6: Sample Fulfillment

① Accept → Draft Order 자동 완료 → ② Sample Sent (마케터 클릭) → ③ n8n poll → Shopify Order 상태 업데이트 → ④ 배송 처리 Sample Shipped → ⑤ Sample Delivered + Content Guidelines 이메일 자동 발송
⑥ 포스팅 안 한 크리에이터에게만 Reminder Sent (선택적)
