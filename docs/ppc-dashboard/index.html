Executor — 9-Stage Pipeline

DataKeeper → ROAS framework → proposals per brand

Pre-flight
DataKeeper Freshness Check
Verify latest >= T-2. If stale, auto-collect 3 channels.
Stage 1-2
Profile & Campaign Data
Brand config + fetch ENABLED campaigns.
Stage 3
Campaign Analysis — ROAS Framework
<1.0→pause | 1.0-1.5→bid-30% | 1.5-2.0→bid-15% | 2.0-2.5→opt-10% | 2.5-3.0→opt | 3.0-5.0→bud+20% | >5.0→bid+10%,bud+30%
Stage 4
Search Term & Keyword Data
DataKeeper (50K limit) → dedup vs execution history.
Stage 5
Simulator + DataForSEO + Analysis
Harvest: ACOS<target, 10+clicks, 1+sale | Negate: $0+high$, ACOS>2x, SIM($10+) | Bids: strong+20%, ineff-20%
Stage 6
Keyword Matrix
7d vs 30d trends, cannibalization, Google CPC benchmarks.
Stage 7-8
Cross-Platform + ASIN P&L
Google/Meta/Shopify. Full-cost margin: COGS+FBA+Export+Returns.
Stage 9
Save & Email
HTML email + JSON proposal + budget impact summary.

Backtest Simulator

⚖️ Module 2: Bids

ROAS <2x→bid-20%, >5x→bid+15%. Grosmimi: 4.06→4.59x (+0.53). 17 down, 8 up.

🔗 SIM→Executor

top_waste_terms ($10+ save) → SIM negates. Annotate+sort by savings. Source file logged in email.
