BEFORE (현재)
• LT/HT 같은 스테이지 (계약/결제 없음)
• HIL 표시: Sample Sent만 (LT), Draft Ready+Replied+Needs Review+Sample Sent (HT)

AFTER (수정 후)
• LT: Draft Ready → ... → Delivered → Reminder → Posted
• HT: Draft Ready → ... → Send Contract → Contract Signed → Accept → ... → Payment Sent → Posted
• HIL 정확하게 표시 (마케터 액션 필요한 곳만)
